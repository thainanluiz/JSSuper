# JSSUPER CORE

How to install

```shell
npm i jssuper
```

<br>
<br>

Available functions:

- isANumber( value ) :
  <br>
  Return true or false

```js
if(_.isANumber(5)){
    Your code...
} else {
    More code...
}
```

<br>

- randomINT( min, max ) :
  <br>
  Return random number between min and max args

```js
console.log(_.randomINT(0, 10));
```
